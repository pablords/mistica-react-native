# mistica-react-native

framework that contains reusable UI components and utilities

## Installation

```sh
npm install mistica-react-native
```

## Usage

```js
import { MisticaTextInput } from "mistica-react-native";

// ...
    <MisticaTextInput
        style={{
          position: 'absolute',
          top: 100,
          left: 15,
          right: 0,
          bottom: 0,
          width: '90%',
        }}
        inputText="text"
        onChangeText={() => {}}
    />
```
